## 必要准备

- 一个github账号
> 没了，你以为这很难？ 跟着教程一步步来，很简单的

## fork仓库

- 前往网页的[仓库](https://github.com/Linking-ET/linking-et.github.io)

- 点击右上方的[fork](https://github.com/Linking-ET/linking-et.github.io/fork)按钮，创建一个属于你自己的分支

- 等待一会，github会自动完成分支
